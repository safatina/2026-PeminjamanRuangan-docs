# 2026-PeminjamanRuangan Docs
# Struktur Dokumentasi
1. arsitektur/
  Folder ini berisi diagram dan penjelasan arsitektur aplikasi, termasuk alur data, komponen backend dan frontend, serta integrasi dengan database.
2. panduan/
  Berisi panduan teknis untuk menjalankan dan memelihara proyek:
  - cara-run-docker.md – Petunjuk menjalankan aplikasi menggunakan Docker dan Docker Compose.
  - troubleshooting.md – Tips menyelesaikan masalah umum saat menjalankan proyek.
3. api-spec.md
  Dokumentasi spesifikasi API, termasuk endpoint, metode HTTP, format request/response, dan contoh penggunaan.
# Cara Menggunakan Dokumentasi
1. Buka folder arsitektur/ untuk memahami alur sistem.
2. Ikuti panduan di panduan/cara-run-docker.md untuk run Docker.
3. Jika terjadi error atau kendala, cek panduan/troubleshooting.md.
4. Gunakan api-spec.md untuk referensi pengembangan frontend atau integrasi API.
